📖編輯說明 · 給組員的使用手冊

檔案怎麼開？

推薦工具（任選）

  • VS Code（免費下載、最推薦）：code.visualstudio.com
  • Sublime Text（免費試用）
  • Mac 內建「文字編輯」app（最簡單、但沒語法 highlight）
  • 不要用 Word、Pages、Google Docs 開 HTML（會破壞格式）

步驟

  1. 在 Finder 找到 Documents/oa-geo-strategy-v6.7/ 資料夾
  2. 進去 tasks/ 資料夾、看你要改的任務檔（例如 01-wikipedia.html）
  3. 右鍵 → 開啟方式 → 選 VS Code（或其他文字編輯器）
  4. 找到 <!-- ✏️ 可改 --> 標記的段落、直接改文字
  5. 存檔（Cmd+S）
  6. 回到瀏覽器、按 Cmd+R 重新整理就看得到
✏️

哪些可以改？

  • ✅ HTML 內標有 <!-- ✏️ 可改 --> 的段落（純文字）
  • ✅ 表格內的文字 / 數字
  • ✅ 任務上的主責下拉（綠色追蹤區、自動儲存）
  • ✅ 進度滑桿（綠色追蹤區、自動儲存）
  • ✅ 備註欄（綠色追蹤區、自動儲存）
  • ✅ 修改記錄區（橘色虛線框）

🚫 不要改

  • ❌ <style> 區段的 CSS（會壞掉）
  • ❌ <script> 區段的 JavaScript
  • ❌ shared/ 資料夾內的檔案（styles.css / state.js）
  • ❌ <a href="..."> 內的連結（會跳錯地方）
  • ❌ <img src="..."> 圖片來源
口訣：看得懂的中文就可以改、看不懂的英文 / 符號不要碰。改之前 Cmd+D 複製一份備份。
🔄

進度共享（Google Drive）

每個任務頁的綠色區塊有 主責 / 狀態 / 進度 / 備註、會自動存在你電腦的瀏覽器裡。但其他人看不到、需要手動分享。

分享步驟

  1. 在任務頁綠色區塊下方按「💾 下載狀態（給其他人）」
  2. 會下載一個 oa-geo-state-2026-XX-XX.json 檔
  3. 把這個檔上傳到 Google Drive 共享資料夾（建議叫「GEO 戰略進度」）
  4. 其他人從 Google Drive 下載最新的 JSON 檔
  5. 在任務頁綠色區塊下方按「📂 載入他人狀態」、選剛下載的 JSON 檔
  6. 頁面會自動重整、顯示最新進度
建議節奏：每週開會前一天、由 Posh 下載最新狀態、組員根據 Posh 的版本載入、避免衝突。如果同一週多人都改了、以 Posh 那份為準。
📊

Dashboard 怎麼用？

點 Dashboard 頁 可以一覽全 11 任務的進度、誰負責、狀態、最後更新日。Posh 主要看這頁掌握全局。

  • 🟢 完成的任務會變淡、底色轉綠
  • 🔴 卡住的任務會紅色標記
  • 📝 有備註的任務會在卡片下方顯示
  • 點任務卡片 → 跳到該任務細節頁
⚠️

常見問題

Q：我改了但其他人看不到？

A：狀態存在你電腦的瀏覽器、別人看不到。要按「下載狀態」→ 上傳 Google Drive → 對方下載 → 載入。

Q：我換電腦看不到進度？

A：同上、需要從 Google Drive 載入。或在原電腦下載 JSON → 新電腦載入。

Q：我改壞了怎麼辦？

A：① 沒存檔就 Cmd+Z 還原 ② 存檔了但網頁顯示怪、找 Posh 求救 ③ 進度狀態壞了、按「🗑 重置全部狀態」回到預設

Q：可以加新任務嗎？

A：先跟 Posh 討論。要動 state.js + 加新 HTML、不是改改文字就好。

Q：怎麼列印 / 匯 PDF？

A：在瀏覽器按 Cmd+P → 另存為 PDF。每個任務頁都可以單獨列印。

協作版

🎯 一頁摘要 · 給組員理解方向

核心論述：橘子蘋果已有多組可公開查證的權威資產與教學成果，但目前分散在媒體、學術、政府、獎項、GBP、影片與官網各處。協作版的目的，是先把這些素材整理成可查證、可引用、可交辦的證據包。

3 個協作方向：

  • 權威信任：政府 / 國際 / 學術 / 獎項證據整理成官網可用素材。
  • 真實口碑：家長故事、KOL 候選、評論入口與揭露規範都先合規化。
  • 內容資產：影片、文章、Wiki 候選來源與 PR 素材先整理成可搜尋的結構。
為什麼現在做？因為 LLM / Google 搜尋正在改變家長蒐集補習班資訊的路徑；公開、可查證、可信任的來源會越來越重要。組員先把資料整理乾淨，管理層才能決定哪些搬到官網、哪些拿來提案、哪些暫時保留。

3 個不可踩的紅線：

  • 不做假口碑、假帳號或任何難以揭露的推薦。
  • 不把未立案或不符合平台規則的地址做成獨立 GBP。
  • 不把登入牆、內部文件或口頭記憶寫成公開可查證事實。

組員先做：

  • 補公開連結、截圖、來源日期、可引用段落。
  • 把待高層確認、待法規確認、待預算確認的事項標清楚。
  • 每週只回報「已補證據、卡住原因、下一步」。

  • 未經管理層確認就承諾新課程、新合作或新預算

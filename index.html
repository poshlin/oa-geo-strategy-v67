CEO 提案總綱

業績框架

不從 GEO 角度提、從「招生轉換率 + 業績槓桿」角度。CEO 業績導向、必須讓他聽到 ROI。

🎯 一頁簡報 · 給 CEO

核心論述：橘子蘋果 14 年累積已有大量被低估的權威資產（HolonIQ 2023 EdTech 50 跟 BenQ 同級、iLink 通訊翟本瑞獨立深度報導、5 篇學術論文真實引用、品保協會履保會員、2,244 條 4.9★ GBP 評論），但 90% 都沒拿出來用。提案不是「做新東西」、是 「把已有的變成招生轉換率工具」+「補強會直接影響家長選擇的 3 個槓桿」。

3 個業績槓桿（直接影響家長報名）：

  • 權威信任分：政府 / 國際 / 學術背書放官網 + 業務話術 → 家長從「比較中」選橘蘋（任務 5+10+11、預算 30-35 萬）
  • 真實口碑放大：家長 LINE 群 UGC 自動化 + 親子部落格體驗（合規揭露）+ Miula 模式知識型 KOL（ROI 13.8x 已驗證）→ 家長「猶豫中」被推一把（任務 3+8、預算 100 萬）
  • 內容資產 SEO/GEO：影片 4 軌制 + Medium 重啟（已有文章被學術引用）+ Wiki 條目 + PR 深度報導 → 家長「主動搜尋」找得到橘蘋（任務 1+2+4+9、預算 115-150 萬）
為什麼現在做？因為 LLM / Google 搜尋持續吞食家長注意力、ChatGPT 推薦哪家補習班會決定 30% 的家長選擇、權威背書是 LLM 引用率的關鍵。HolonIQ 證明我們已是業界前 50、但這個資產在我們官網完全不見、業務不講、Wiki 沒寫 — 對手只要把這個拿走、我們就輸了。

3 個避雷（已 deep-research 驗證）：

  • 不簽論壇口碑年約（三星寫手門 1,000 萬罰款先例、戰國策 / 貳肆加全踩線）
  • 不在週末租場建獨立 GBP（補習班法 5-25 萬罰 + Google 政策下架）
  • 不純做業配稿（LLM 對純業配降權、改深度報導）

3 件不開發、3 件全推：

  • 不開發新校用課程、新人力（用既有 5 角色）、新代操工具（第一年）
  • 全推權威曝光（任務 5+10+11）、UGC 自動化（任務 3）、內容資產（任務 9）

第一季可見成果：HolonIQ 徽章 + 5 篇學術 + iLink + 北市酷課雲 + 拾穗 = 3 個月內官網信任區轉換率拉抬可測量（A/B test）、業務話術升級 + 招生轉換率提升 X%。

預算建議：CP 最高方案 253-292 萬/年、第一季先投 25%（63-73 萬）驗證、Q2 起根據招生轉換率數據加碼。
